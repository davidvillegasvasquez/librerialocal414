Comandos httpie para usar los simplejwt: 

Para acceder a una vista protegida con Simple JWT en Django REST Framework utilizando HTTPie, debes seguir dos pasos principales: obtener el token de acceso y luego usarlo en el encabezado de autorización (Authorization Header).

1. Obtener el Token de Acceso (POST)
Primero, envía tus credenciales (usuario y contraseña) al endpoint de autenticación (generalmente /api/token/) para obtener el token access:

http POST http://127.0.0.1:8000/api/token/ username=david password=chacha01

La respuesta será un JSON que contiene un token access y un refresh. Copia el valor del campo access:

{
    "access": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
    "refresh": "..."
}

1.1 Para refrecar el token:

http POST http://localhost:8000/api/token/refresh/ refresh="eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ0b2tlbl90eXBlIjoicmVmcmVzaCIsImV4cCI6MTc3NzQxOTc0NCwiaWF0IjoxNzc3MzMzMzQ0LCJqdGkiOiJiMmYxZmE1NjBlMTI0M2MxYTRmMTg5ODU5MTc1YTUyYSIsInVzZXJfaWQiOiIxIn0.WTRhaX59mDsXcZN6u_DhMbIsSzP_KcAd-niQeTZFf5Q"


2. Acceder a la Vista Protegida (GET)

Utiliza el token access obtenido en el paso anterior en el encabezado de autorización. El formato requerido es Bearer <tu_token>. 

http GET http://127.0.0.1:8000/api/vista-restringida/ "Authorization: Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..."

Métodos Alternativos y Tips
Uso de comillas: Asegúrate de usar comillas dobles " alrededor del encabezado Authorization: Bearer ... para evitar problemas con caracteres especiales del token en la terminal.

Guardar el token en una variable: Para mayor comodidad, puedes guardar el token en una variable:

TOKEN=$(http POST http://127.0.0.1:8000/api/token/ username=usuario password=pass | jq -r .access)

http GET http://127.0.0 "Authorization: Bearer $TOKEN"

Plugin de HTTPie: Existe un plugin llamado httpie-jwt-auth que simplifica la sintaxis, permitiendo usar http --auth-type=jwt --auth=token_aqui :8000/api/ruta/.

        ***

Guardar token de refrescamiento en una variable de terminal:

export REFRESH_TOKEN=$(http POST http://localhost:8000/api/token/ username=david password=chacha01 | jq -r .refresh)

Para usar el token guardado en la variable de terminal:

http POST http://localhost:8000/api/token/refresh/ refresh=$REFRESH_TOKEN
